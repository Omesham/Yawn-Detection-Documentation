ANACONDA INSTALLATION FOR JUPYTER NOTEBOOK

Step 1: Download Anaconda: click link Anaconda Download Click on get additional installers for other operating system options

Step 2: Click on the executable file and install Anaconda, select the path where you want it installed and click next and follow the installation process prompts.It takes some time so don't panic.

Step 3: Go to directory or folder where it is installed, you should see Anaconda3 directory/folder created by the installation.

Step 4: (Set up environment)

Step 5: (Copy required directory paths and add new environment)

Now copy the Anaconda File Path, click on your start menu and search for "environment"

Step 6: Open environment window and add these directories paths as seen in the image below.

Step 7: Now launch your anaconda terminal:click start menu and search for "Anaconda" open terminal and tpe the command to create an environment to install necessary packages because you can't install these packages in your base directory
NOTE! The packages need this environment you are about to create to work with your packages, when you shut down your terminal you will always be required to activate your environment to use these packages so make sure to remember your environment or type " conda env list " without the doube quote to show list of environments env to show your environment

Step 8: (create environment) Type command: conda create -n myenv (myenv is whatever name you give your environment) and hit enter copy or type conda activate myenv (to activate environment) see image below

Step 9:

Run these commands to install conda packages the necessary packages you need are a. dlib

b. python

c.numpy

d.opencv

e. jupyter notebook

Now if in the future you want to activate more packages in this environment , activate the environment ( conda activate myenv and repeat these processes)

Step 10:

Type commands and hit enter, type y to continue always commands: conda update conda conda update anaconda conda install -c conda-forge dlib

conda install -c conda-forge python

conda install -c conda-forge numpy

conda install -c conda-forge opencv-python

conda install -c conda-forge jupyter notebook

Step 11: After installing packages Type jupter notebook to launch jupyter notebook. You can run your code on jupyter notebook now. The code will execute and launch 2 window frames, one for you face landmarks and the other the live webcam where you can yawn and it counts it. Now I hardcoded my lip distance, feel free to play with this figure to adjust sensitivity level, because you don't want it to count when you're talking as a Yawn.
